2. Photos in Different Formats

JPEG Format

Explanation: This image is in JPG format. JPG is a lossy format (DCT algorithm) ideal for digital photography because it compresses complex colors well.

HDR (High Dynamic Range) Photo Simulation

Explanation: This represents an HDR image. HDR provides higher contrast than standard dynamic range (SDR), preserving details in both the darkest and brightest areas. It requires HDR-compatible devices to view its full potential.

3. Graphics in Different Formats

PNG Format

Explanation: This is a PNG graphic. PNG is a lossless format (DEFLATE algorithm) that supports 24 or 32-bit graphics and is highly suitable for images requiring transparency.

SVG (Scalable Vector Graphic)

Explanation: This is an SVG graphic. SVG is an XML-based vector image format. Because it is vector-based (mathematical shapes), it scales perfectly without losing quality, reducing the amount of data needed to transfer.
